Instalar XAMPP en Solus 4.3

27, 2020 Linux

Solus es una distribución de Linux fantástica: Si bien no tiene tantos paquetes en sus repositorios, es tremendamente rápida (aún mas que Manjaro a mi parecer), además de muy atractiva debido a Budgie.

Sin embargo, en ocasiones puede ser un dolor de cabeza lidiar con Solus, debido a lo “artesanal” y exótico de la distribución.

Una de esas cosas que en otras distribuciones es simple de hacer, pero requiere pasos adicionales en Solus, es instalar XAMPP. En esta entrada explicaré los pasos necesarios para ejecutar XAMPP en la última versión de Solus (4.1)

Para instalar XAMPP, hay que bajar el instalador desde Apache Friends, abrir una terminal, navegar hasta la carpeta donde descargamos el instalador y ejecutar el instalador con permisos de super usuario:

sudo ./xampp-linux-x64-7.1.11-0-install. run

Sin embargo, antes de poder ejecutar XAMPP, es necesario modificar algunos archivos de configuración, de lo contrario, al iniciar XAMPP obtendremos una serie de errores al momento de intentar iniciar todo.

Después de instalado XAMPP, hay que abrir el archivo httpd.conf con el editor de su preferencia:

sudo pluma /opt/lampp/etc/httpd.conf

Buscar las líneas:

User daemon
Group daemon

Y reemplazar daemon por nobody, para que quede de la siguiente manera:

User nobody
Group nobody

Guardamos cambios y después, abrimos el archivo proftpd.conf

sudo pluma /opt/lampp/etc/proftpd.conf

Buscamos las líneas:

# Set the user and group that the server normally runs at.
User daemon
#Group daemon

Y reemplazamos User daemon por User nobody, de tal manera que el archivo quede así:

# Set the user and group that the server normally runs at.
User nobody
#Group daemon

Y listo, con eso XAMPP estará listo para usarse.
